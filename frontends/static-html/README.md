# Static HTML Frontend Blueprint

Pure HTML/CSS/JS with Tailwind CDN — no build step needed. For simple sites.

## Usage
```bash
cp -r frontends/static-html/* ~/projects/<name>/frontend/
# Just serve the HTML files — no build needed
```

## What's Included
- `index.html` — Home page with dark theme
- `css/style.css` — Custom CSS with Prince's theme
- `js/main.js` — Basic interactivity
- `components/` — Reusable HTML snippets (nav, footer, card)

## Prince's Theme (Pre-configured)
- Dark background: `#0a0a0f`
- Glassmorphism via CSS
- Gradient accents: indigo → purple
- Tailwind CSS via CDN
- Responsive: mobile-first
